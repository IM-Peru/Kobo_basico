1. ¿Que es Kobo?


Kobo Toolbox es una plataforma para la recolección de datos en el sector humanitario

  • Sitio web: https://www.kobotoolbox.org/

  • Video de introducción: https://www.youtube.com/watch?v=4PNtT51h3CQ


Esta plataforma fue desarrollada para fortalecer:

  • Precisión y eficiencia,

  • Coordinación,

  • Calidad

de los datos recolectados.

La tecnología de Kobo es open source y completamente gratuita para el uso humanitario.

Antes de empezar



Cuando quieran empezar una recolección de datos, piensen siempre bien en los siguientes criterios:

  • ¿La encuesta incluye datos personales?

  • Si la encuesta incluye datos personales, ¿cómo los estoy protegiendo?

  • ¿Cómo voy a respectar los criterios de protección de las personas en mi encuesta? ¿Tengo activadas rutas de protección?

  • ¿Estoy logrando los objetivos que necesito con la encuesta? ¿Es necesario pedir más información?

  • ¿He formulado las preguntas/categorías de respuesta de manera correcta?

  • ¿He seleccionado la metodología correcta para la recolección de datos?

Formato xls: Qué es y por qué utilizarlo?

El diseño de cuestionarios en formato web tiene límites.

Este formato no es óptimo para desarrollar cuestionarios con un gran número de preguntas - y no es posible aprovechar todas las funcionalidades de Kobo.

Para esto, se recomienda manejar los cuestionarios en formato xls.

Pueden descargar el formato xls del cuestionario que acaban de crear directamente desde Kobo:

Formato xls: Estructura básica

El formato xls se abre en Excel y automaticamente tiene un mínimo de tres pestañas:

- “Survey”

  • Estructura del formulario y lista completa de preguntas;

  • Una fila = una pregunta o un cambio en formatación;

- “Choices”

  • Lista de opciones para preguntas de selección multiple en el formulario;

  • Una fila = una opción

- “Settings”

  • Información general sobre el formulario: nombre, ID e idioma

Formato xls: Pestaña “Survey”

La pestaña “Survey” incluye un mínimo de tres columnas obligatorias:



  • type: El tipo de comando/pregunta que se está dando para esta fila del formato

  • name: Un ID único para la linea

  • label: La pregunta que va a aparecer a la persona que llena el formulario

Formato xls: Pestaña “Choices”

La pestaña “Choices” contiene las listas de opciones multiples del formulario.

También se incluyen un mínimo de tres columnas obligatorias:



  • list_name: se refiere al grupo de opciones para una respuesta multiple
  • name: Un ID único para la opción
  • label: Como la linea va a aparecer a la personas que llenará el formulario

Formato xls: Tipos de preguntas

Kobo permite un gran número de tipos de preguntas:

Formato xls: Columnas opcionales en “Survey”

La pestaña “Survey” tiene tres columnas obligatorias, pero se pueden añadir más dependiendo de las características de las preguntas.

Algunos ejemplos:

  • Hint

  • Constraint & constraint_message

  • Relevant

  • Calculation

  • Required

  • Group

  • Repeat

  • Languages

  • Appearance

Descargar el formulario

Una vez configurado el aplicativo, pueden descargar el formulario seleccionando “Get Blank Form”:

Atención: El formulario descargado no se sincroniza automaticamente en caso de cambios en el formulario en Kobo! Si hicieron cambios (por ejemplo, revisando el formato xls) después de haber desplegado un formulario, tienen que descargar la nueva versión.

Ingresar datos

Para ingresar datos, pueden seleccionar “Fill Blank Form” y llenar el formulario.

Atención: El formulario no es enviado automaticamente. Para enviar el formulario, regresen al menú principal y seleccionen “Send Finalized Form”.
